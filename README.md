# ESEI Student Portal

This is the ESEI Student Portal single-page application, built with **Next.js 15**, **Tailwind CSS**, and **ElevenLabs AI Agents**.

## Features

- **Login Screen**: Enter credentials (default values provided for testing).
- **Dashboard**: Overview of courses, stats, and AI Tutor.
- **AI Tutor**: Integrated ElevenLabs Conversational AI Agent for homework assistance.
- **Grading Pathway**: Visual pathway of grades and feedback.
- **File Upload**: Submit assessments.

## Technologies

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Voice AI**: ElevenLabs ConvAI
- **Deployment**: Vercel

## Deployment Status

This project is configured for seamless deployment on Vercel.
Ensure your Vercel project settings have the "Framework Preset" set to **Next.js**.

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.
