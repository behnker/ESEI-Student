# ESEI Student Portal

This is the ESEI Student Portal single-page application.

## How to Run

Simply open `index.html` in your web browser.

## Features

- **Login Screen**: Enter credentials (default values provided for testing).
- **Dashboard**: Overview of courses, stats, and AI Tutor.
- **Grading Pathway**: Visual pathway of grades and feedback.
- **File Upload**: Submit assessments.

## Technologies

- **HTML5**
- **Tailwind CSS** (via CDN)
- **Lucide Icons** (via CDN)
